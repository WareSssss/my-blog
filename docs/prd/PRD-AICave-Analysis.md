# AI树洞 (AICave) 产品需求文档 (PRD)

## 1. 产品概述
AI树洞是一个集 AI 资讯聚合、多平台文章收录、多媒体展示及 AI 辅助工具于一体的综合性 AI 内容社区平台。它通过自动化采集技术，为用户提供一站式的 AI 行业动态、深度文章及创意作品。

## 2. 目标用户
- **AI 开发者/从业者**：获取最新行业资讯、论文摘要及技术趋势。
- **内容创作者**：寻找灵感、展示 AI 生成的作品。
- **AI 爱好者**：探索 AI 应用、体验智能体交互。

## 3. 功能模块详情

### 3.1 概览控制台 (Overview Dashboard)
- **数据看板**：实时展示平台内容总量（资讯、文章、图片、视频）。
- **平台分布分析**：以饼图或列表形式展示收录文章的平台来源（小红书、知乎、微信、X/Twitter、Reddit）。
- **快速入口**：提供核心工具（MD转图片、AI生图）及社区加入的快捷访问。

### 3.2 AI 资讯 (AI News/Hotspots)
- **多源采集**：每 2 小时自动从 Twitter、36氪、AIBase 等渠道抓取最新 AI 动态。
- **时效过滤**：支持按时间段（如 2 小时、4 小时等）筛选热点资讯。
- **来源标注**：清晰标注资讯来源及原始链接，支持点击跳转。

### 3.3 AI 文章 (AI Articles)
- **多平台聚合**：深度收录来自主流平台的 AI 专题文章。
- **内容清洗**：对采集到的文章进行格式化处理，提升阅读体验。
- **搜索与分类**：支持按关键字、平台、标签进行文章检索。

### 3.4 创意多媒体 (Multimedia Gallery)
- **AI 图片**：展示创作者公开的 AI 生成作品，支持流式加载。
- **AI 视频**：汇集前沿 AI 视频案例，提供在线预览。

### 3.5 AI 辅助工具 (AI Tools)
- **MD 转图片**：支持将 Markdown 文本快速转化为美观的分享图片。
- **AI 生图**：集成主流生图模型（如 Stable Diffusion/Midjourney 等），提供在线提示词出图功能。

### 3.6 Agent 智能体 (Interactive Agents)
- **智能对话**：提供不同领域的垂直 Agent，支持特定任务的对话交互。
- **工具调用**：Agent 可集成搜索、分析等扩展能力。

## 4. 业务流程梳理

### 4.1 内容流转流程
1. **自动采集**：定时脚本从外部平台（API/爬虫）获取原始数据。
2. **结构化处理**：对原始 HTML/JSON 进行提取，转化为统一的 DTO 模型。
3. **内容存储**：持久化至数据库，并根据配置触发图片转存（OSS）。
4. **前端分发**：用户通过 Web 端按分类、热度查看内容。

### 4.2 智能体交互流程
1. **用户提问**：通过 Chat 界面输入需求。
2. **意图识别**：后端 LLM 识别用户意图。
3. **工具执行**：根据意图调用本地函数或 RAG 知识库。
4. **流式响应**：以 SSE 或 WebSocket 形式返回回复。

## 5. 系统边界与外部依赖
- **外部平台**：Twitter (X), 36Kr, AIBase, 小红书, 知乎, Reddit。
- **基础设施**：
  - **数据库**：PostgreSQL (关系型数据), Qdrant (向量检索)。
  - **存储**：阿里云 OSS / 腾讯云 COS (图片/视频存储)。
- **AI 模型**：OpenAI (GPT-4o), DeepSeek (V3/V4), Stable Diffusion (生图)。

## 6. 非功能性需求
- **采集频率**：核心资讯 2 小时一更。
- **性能指标**：首屏加载 < 2s，图片采用 WebP 压缩。
- **安全性**：敏感操作（如生图、Agent 调用）需 Token 校验及频率限制。
