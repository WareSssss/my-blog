# 深度学习基础：从神经元到反向传播

理解 AI Agent 的“思考”本质，需要对神经网络的基本原理有直观的认识。本篇将带你拆解深度学习的核心“黑盒”。

---

## 1. 神经元：计算的原子
神经网络是由大量相互连接的**神经元**（Neurons）组成的。
- **输入 (Input)**: 接收到的信息（如文字的向量）。
- **权重 (Weights)**: 代表该信息的重要性。
- **偏置 (Bias)**: 调整神经元被激活的难易程度。
- **激活函数 (Activation Function)**: 决定该神经元是否向后传递信号（如 ReLU, Sigmoid）。

> **公式**: $Output = Activation(\sum(Input \times Weight) + Bias)$

---

## 2. 前向传播 (Forward Propagation)
数据从输入层开始，一层层经过神经元的计算，最终在输出层得到预测值（例如预测下一个 Token 是什么）。

---

## 3. 损失函数 (Loss Function)：衡量差距
损失函数（如交叉熵损失）用于计算 **预测值** 与 **真实目标值** 之间的距离。
- 损失值越大，说明模型预测得越不准。
- 我们的目标是最小化这个损失。

---

## 4. 梯度下降 (Gradient Descent)：寻找最优解
想象你在山上，目标是走到山谷（损失最小的地方）。
- **梯度**: 山坡最陡的方向。
- **步长 (Learning Rate)**: 你每一步迈多大。
- 梯度下降就是沿着坡度最陡的方向反向走一步，逐步逼近山谷。

---

## 5. 反向传播 (Backpropagation)：责任分配
这是深度学习最天才的地方。
1. 计算最后的损失。
2. 从后往前，根据损失的大小，计算每一个权重（Weight）对误差的贡献。
3. 根据贡献大小，按比例更新权重。
- **口诀**: “谁错得多，就改谁的权重”。

---

## 6. 学习建议
- **直观理解**: 重点理解“权重更新”的逻辑，不需要手写求导公式。
- **关联实战**: 思考在本项目中，大模型在回答问题时，本质上是在其数十亿个预训练好的“权重”中寻找最匹配输入指令的信号通路。
